# SplitwiseApp
- Splitwise is a app for friends and roommates to track bills and other shared expenses, so that everyone gets paid back.
# Compatibility
- This project is written in Swift 4.2 and Xcode 10.1
# Tools
- RxSwift
- Realm
- MVVM
